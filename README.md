# Java Chatbot

## Description

This project is a simple console-based chatbot developed in Java. The application interacts with the user by asking questions, performing basic calculations, and testing programming knowledge through a multiple-choice quiz.

## Features

* Greets the user and asks for their name
* Calculates the user’s age using mathematical logic
* Counts from 0 to a given number
* Includes a multiple-choice quiz about programming
* Repeats the question until the correct answer is selected

## How to Run

```
javac src/bot/ChatBotMain.java
java -cp src bot.ChatBotMain
```

Beschreibung

Dieses Projekt ist ein einfacher, konsolenbasierter Chatbot, der in Java entwickelt wurde. Die Anwendung interagiert mit dem Benutzer, indem sie Fragen stellt, grundlegende Berechnungen durchführt und das Programmierwissen durch ein Multiple-Choice-Quiz testet.

Funktionen
Begrüßt den Benutzer und fragt nach seinem Namen
Berechnet das Alter des Benutzers mithilfe mathematischer Logik
Zählt von 0 bis zu einer angegebenen Zahl
Enthält ein Multiple-Choice-Quiz über Programmierung
Wiederholt die Frage, bis die richtige Antwort ausgewählt wird
Ausführung
javac src/bot/ChatBotMain.java
java -cp src bot.ChatBotMain
